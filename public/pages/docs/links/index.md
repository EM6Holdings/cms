# Markdown or HTML URL replacement

Sphido replace &#123;url} with real server URL, follow some examples:
 
| Code in HTML or Markdown        | Result
|---------------------------------|----
| &#123;url}                      | {url}  
| &#123;url aaa}                  | {url aaa}  
| &#123;url /examples/email/send} | {url /examples/email/send}
